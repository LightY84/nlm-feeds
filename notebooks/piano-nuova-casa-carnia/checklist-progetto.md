# Checklist Progetto — Nuova Casa in Carnia

Roadmap completa dalle prime valutazioni all'abitabilità. Spunta le voci man mano.

---

## Fase 0 — Definizione obiettivi

- [ ] Definire budget massimo complessivo (acquisto + costruzione + imprevisti 10-15%)
- [ ] Scegliere strategia: nuova costruzione / acquisto + ristrutturazione / acquisto rudere e ricostruzione
- [ ] Definire requisiti: mq, camere, garage, orientamento, vista, esposizione solare
- [ ] Decidere comune o area preferita (Tolmezzo, Arta Terme, Ovaro, Paluzza, Sutrio, Forni di Sopra, ecc.)
- [ ] Valutare distanza da servizi: ospedale (Tolmezzo), scuole, supermercati, lavoro

## Fase 1 — Ricerca terreno / immobile

- [ ] Verificare destinazione urbanistica sul PRGC del comune (zona edificabile)
- [ ] Richiedere Certificato di Destinazione Urbanistica (CDU) al comune
- [ ] Verificare indice di edificabilità e vincoli (paesaggistico, idrogeologico, sismico)
- [ ] Controllare classificazione zona sismica (la Carnia è in gran parte zona 1-2)
- [ ] Verificare rischio frane/valanghe/esondazioni sul piano PAI regionale
- [ ] Verificare presenza e costi di allacciamenti: acqua, luce, gas/GPL, fibra, fognatura
- [ ] Valutare accessibilità invernale (neve, pendenza strada, esposizione nord)
- [ ] Far fare perizia geologica preliminare se il terreno è in pendenza

## Fase 2 — Progettazione

- [ ] Selezionare 2-3 professionisti (architetto/geometra/ingegnere) e confrontare preventivi
- [ ] Progetto preliminare con stima costi
- [ ] Scegliere tecnologia costruttiva: muratura tradizionale / legno (X-lam, telaio) / misto
- [ ] Definire classe energetica obiettivo (consigliata A4 / casa passiva, dato il clima montano)
- [ ] Progettare riscaldamento: pompa di calore + fotovoltaico / pellet-legna / ibrido
- [ ] Relazione geologica e geotecnica definitiva
- [ ] Progetto strutturale antisismico (obbligatorio, depositato presso la Regione)
- [ ] Pratica edilizia: Permesso di Costruire (PdC) o SCIA alternativa
- [ ] Eventuale autorizzazione paesaggistica se in zona vincolata

## Fase 3 — Finanziamento

- [ ] Confrontare mutui (almeno 3 banche + broker)
- [ ] Verificare contributi regionali FVG prima casa (la Regione ha storicamente contributi specifici per aree montane)
- [ ] Verificare bonus edilizi statali vigenti e detrazioni
- [ ] Verificare agevolazioni per under 36 / prima casa se applicabili
- [ ] Mettere a budget oneri di urbanizzazione e costo di costruzione comunali

## Fase 4 — Cantiere

- [ ] Gara tra almeno 3 imprese con computo metrico identico
- [ ] Verificare DURC e referenze delle imprese
- [ ] Contratto di appalto con penali su tempi e SAL definiti
- [ ] Nomina direttore lavori e coordinatore sicurezza
- [ ] Scavi e fondazioni
- [ ] Struttura grezza
- [ ] Copertura (dimensionata per carico neve — in Carnia è rilevante)
- [ ] Impianti (elettrico, idraulico, termico, ventilazione meccanica controllata)
- [ ] Cappotto / isolamento
- [ ] Finiture interne ed esterne

## Fase 5 — Chiusura

- [ ] Collaudo statico
- [ ] Certificazioni impianti (DM 37/08)
- [ ] Attestato di Prestazione Energetica (APE)
- [ ] Segnalazione Certificata di Agibilità
- [ ] Accatastamento (DOCFA)
- [ ] Voltura utenze e residenza
